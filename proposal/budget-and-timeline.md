# Budget And Timeline

## Budget Summary

This is a six-month research budget for a documentation-first, simulation-first project. It is intentionally sized for a focused grant effort rather than a production platform build.

| Category | Estimated Cost | Notes |
| --- | ---: | --- |
| Compute | $7,500 | Offline simulations, batch evaluations, and reproducible benchmark runs |
| API credits | $9,000 | Controlled model evaluation, prompt comparisons, and agent-role experiments |
| Infrastructure | $4,500 | Artifact storage, documentation hosting, versioned datasets, and local development support |
| Research and documentation | $24,000 | Dataset design, scenario authoring, threat modeling, benchmark writing, and publication |
| Total | $45,000 | Proposed grant request |

## Timeline

### Month 1

- finalize repository structure and guardrails
- publish project summary, methodology outline, and threat model
- release initial CEID schema

### Month 2

- author synthetic scenario catalog
- create the first anomaly samples
- define labeling guidance and provenance fields

### Month 3

- formalize simulation pipeline stages
- define narrow agent responsibilities
- create benchmark task specifications

### Month 4

- build a lightweight offline evaluation harness
- test schema coverage and annotation consistency
- review wording for overclaims and scope drift

### Month 5

- run initial benchmark passes
- record limitations and unresolved edge cases
- prepare public release documentation

### Month 6

- publish the first research package
- release synthetic samples, methodology docs, and benchmark definitions
- document next-step research questions without claiming operational deployment
